# Hotel Management System

## 📌 Description
This is a full-stack hotel management system with restaurant food ordering functionality.

## 🚀 Features
- Room booking system
- Customer management
- Restaurant food ordering
- Admin functionalities

## 🛠 Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React (Vite)
- Database: H2 / MySQL

## ▶️ How to Run
1. Start backend (Spring Boot)
2. Navigate to frontend folder
3. Run:
   npm install
   npm run dev
